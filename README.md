# ChatPlugin

# English

A plugin that allows player use keyboard to communicate with teammates.
This plugin send message to same side as player's.

Usage:

Step 1. Press ~
Step 2. input command .say and content
Step 3. Press Enter

And you'll see message.

# Chinese

一个允许你用键盘与队友交流的插件。
这个插件默认发送信息给相同阵营的玩家。

用法:

1. 按 ~
2. 输入 .say 和你想要的内容
3. 按下回车键

然后你就看到了你输入的内容。
